# Ruedata Tets

This project allows you to enter and create lists of pets and upload them to a CRUD API using React.JS

### Functionalities:
1. Create Pets.
2. Delete Pets.
3. Show Info.

### Technologies used:
1. ReactJs.
2. CRUD.
3. Netlify.

### ¿How to run?
    npm start
